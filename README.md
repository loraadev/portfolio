# 🌐 Portfólio — loraadev

Esse projeto consiste num site de portfólio pessoal desenvolvido com foco em responsividade e boas práticas de front-end. A proposta foi aplicar conceitos de **mobile first**, garantindo uma boa experiência tanto em telas menores quanto em desktops.

## 🔗 Acesse o portfolio

👉 [loraadev.github.io/portfolio](https://loraadev.github.io/portfolio/)

## 🎯 Objetivos

- Praticar desenvolvimento responsivo e com boas práticas
- Organizar layouts de forma limpa e profissional
- Simular um portfólio real para uso profissional
